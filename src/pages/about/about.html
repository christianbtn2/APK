Presentacion Ionic


¿Por que utilizar Ionic?

Es uno de los Frameworks JavaScript que más fuerte está pegando en los últimos tiempos, es Ionic un mega Framework para desarrollo de Apps, basado en otro Framework, AngularJS JavaScript. Este Framework ha tomado mucha importancia en el desarrollo de apps móviles por muchos factores de los cuales hablaremos. Pero uno de los factores principales es la posibilidad de hacer aplicaciones súper estéticas. Sin la necesidad de diseñar mayor cosa. Hoy en día Ionic ofrece una mayor cantidad de plugins como lo son PhoneGap y Cordova, estos pueden acceder a alguna capa nativa del dispositivo (Cámara, GPS, etc)

Plugins

PhoneGap: librería para aplicaciones web, en vez de comunicar directamente con la cámara, se comunica con las librerías de PhoneGap por medio de una especie de puente o interfaz. Cordova: Marco de desarrollo móvil de código abierto, framework para el desarrollo de aplicaciones móviles. Se maneja por medio de CLI (Command Line Inteface).

Factores Primordiales

- Ionic es recomendable para las personas que han trabajado anteriormente con AngularJS, HTML5, Cordova, CSS. - Se puede ejecutar el código independientemente de cuál sea la plataforma objetivo.

Otros Terminos

Framework: es una estructura conceptual y tecnológica de soporte definido. Node js: es un entorno de ejecución para JavaScript. Es orientado a eventos que lo hace liviano y eficiente.
